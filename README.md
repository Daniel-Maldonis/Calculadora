# Calculadora Simples

Uma **calculadora simples** feita com **HTML, CSS e JavaScript**. Permite realizar operações básicas como adição, subtração, multiplicação e divisão, com interface limpa e responsiva.

---

## Funcionalidades

- Operações básicas: `+`, `-`, `*`, `/`
- Botão `C` para limpar a tela
- Botão `←` para apagar o último caractere
- Suporte a números decimais
- Tratamento de erros (exibe `Erro` caso a operação seja inválida)
